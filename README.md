# Laravel Option

## Install

* **Thêm vào file composer.json của app**
```json
	"repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/minhbang/laravel-option"
        }
    ],
    "require": {
        "minhbang/laravel-option": "dev-master"
    }
```
``` bash
$ composer update
```
